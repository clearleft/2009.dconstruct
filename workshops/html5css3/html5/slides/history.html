  2. HyperText Markup Language

    • October 1991: HTML Tags — Tim Berners-Lee
    • November 1995: HTML 2.0 — IETF
    • January 1997: HTML 3.2 — W3C
    • December 1997: HTML 4.0 — Strict, Transitional, Frameset
    • December 1999: HTML 4.01

  3. eXtensible HyperText Markup Language

    • January 2000: XHTML 1.0 — Strict, Transitional, Frameset
    • May 2001: XHTML 1.1 — mime-type:application/xml
    • Never: XHTML 2.0

  4. Web Hypertext Application Technology Working Group

    • June 2004: Opera and Mozilla set up the mailing list
    • HTML5 — Web Applications 1.0 & Web Forms 2.0
    • October 2006: Reinventing HTML — Tim Berners-Lee
    • May 2007: W3C HTML Working Group
    • January 2008: HTML 5
